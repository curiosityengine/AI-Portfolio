Cloud Pastebin

Secure, fast, and elegant code sharing.

Tech Stack

Node.js Express MongoDB CSS Grid/Flex

Project Overview

This application allows users to instantly share code snippets or text. It features a custom-built QR code generator for mobile sharing and a unique "Glassmorphism" UI design.

Key Features

  • Password protection for private pastes.
  • Instant QR code generation for every link.
  • Responsive layout for mobile and desktop.
  • One-click copy to clipboard functionality.
